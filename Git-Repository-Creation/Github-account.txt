

🔹 Step 1: Open Terminal in Your Project Folder
- Navigate to the folder where your code lives:
cd path/to/your/project


🔹 Step 2: Initialize Git
Turn your folder into a Git repository:
git init


🔹 Step 3: Stage and Commit Your Code
Add all files:
git add .


Commit them:
git commit -m "Initial commit"


🔹 Step 4: Create a Repository on GitHub
- Log in to your GitHub account.
- Click the + (top right) → New repository.
- Name it (e.g., my-project).
- Leave it empty (don’t add README for now).
- Click Create repository.

🔹 Step 5: Connect Local Repo to GitHub
Copy the repository URL from GitHub (HTTPS is easiest). Example:
https://github.com/YourUsername/my-project.git


Link your local repo:
git remote add origin https://github.com/YourUsername/my-project.git


🔹 Step 6: Push Your Code
Rename your branch to main (GitHub default):
git branch -M main


Push your code:
git push -u origin main


✅ Done! Your code is now stored in your GitHub account. Refresh the repository page and you’ll see your files.
